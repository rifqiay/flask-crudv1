## HOW TO USE

### Buat venv
```bash
python3 -m venv venv
```
### Aktifkan Lingkungan Virtual (Linux/Mac)
```bash
source venv/bin/activate
```
### Aktifkan Lingkungan Virtual (Windows)
```bash
venv\Scripts\activate
```
### Instal Dependensi
```bash
pip3 install -r requirements.txt
```
### Run Project
```bash
flask run
```
